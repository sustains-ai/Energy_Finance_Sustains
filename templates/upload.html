File Information
Supported Formats:
  • NetCDF (.nc, .netcdf) - Recommended format for climate data
  • CSV (.csv) - Tabular data with headers
  • Text (.txt) - Structured text files
  • GRIB (.grib, .grb) - Gridded binary data
  • HDF5 (.hdf5, .h5) - Hierarchical Data Format
File Size Limit:

Maximum file size: 50 MB

Variable Requirements:

For climate analysis, your data should include:

  • Latitude and longitude coordinates
  • Time dimension (for temporal analyses)
  • Climate variables (e.g., temperature, precipitation)
Need Sample Data?
